## Print

Create a new line

```Python
print('First line \n')
print('Second line')
```

Use variables in the print command

```Python
name = 'Harry'
#Either use this method
print('You are a wizard ' + name + '!')
#Or use this method
print('You are a wizard {}'.format(name))
```
